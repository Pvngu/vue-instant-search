# vue-instant-search

A reusable Vue 3 global search modal focused on controlled state, grouped results, and parent-owned business logic.

## Why this package

- Keeps data fetching in parent components
- Supports grouped favorites and results
- Ships with keyboard navigation and accessible row selection
- Easy to drop into existing admin dashboards
